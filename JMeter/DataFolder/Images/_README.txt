_imageInfoMale.csv contains male names and respective image path
_imageInfoMale.csv contains female names and respective image path
_imageInfoAll.csv contains images from both male and female csv files

The images at the beginning of CSV files are obtained form https://randomuser.me/photos.
The names for these images are randomly generated based on the respective gender.

All images obtained from http://vis-www.cs.umass.edu/lfw/ have prefix 2_ (e.g. 2_Aaron_Eckhart_0001.jpg)
The gender for these images was determined using the first name via query against http://genderize.io/
